# react layout 窗口管理方案

## 起源

有很多并不特别相关的功能 ，然后把每个功能做成一个单独 panel 时，不好决定位置。

最可靠解决方案可能是把选择权给使用者 ?

## 功能

- [ ] 窗口可拖动改变位置，可改变大小
- [ ] 窗口可以关闭
- [ ] 窗口可以拖出来
- [ ] 窗口可以放在其他窗口标题栏成为并列窗口
- [ ] 可以创建窗口容器，里面可摆放窗口或窗口容器
- [ ] 并排窗口标题超过时，显示下拉菜单用于切换窗口
- [ ] 窗口（或窗口容器） 可区域内全屏
- [ ] 窗口（或窗口容器） 可完全全屏
- [ ] 比没有好一点的动画效果
- [ ] 可以导出布局（JSON）
- [ ] 导入布局(JSON)
